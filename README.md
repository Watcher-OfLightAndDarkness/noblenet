# NobleNet

A fun feudal-themed social network where users hold ranks, earn points, and interact with nobles.  

## Features

- User registration and authentication
- Rank system: Citizen → Emperor
- Daily stipends for eligible users
- Points system with upvotes/downvotes
- Admin/Emperor actions: grant points, set ranks, crown Emperor
- Searchable user directory
- Profile avatars and stats
- Action panel for selected users
- Mobile-friendly interface

## Tech Stack

- **Backend:** Node.js, AdonisJS
- **Frontend:** HTML, CSS, JavaScript
- **Database:** MySQL
- **Authentication:** scrypt hashing
- **Utilities:** Luxon for date handling, UUID for unique IDs
